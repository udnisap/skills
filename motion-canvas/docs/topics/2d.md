# @motion-canvas/2d

2D renderer: shapes, text, layout, paths. Provides `makeScene2D` and all built-in 2D nodes.

Back to [index](index.md).

---

## Role

- **makeScene2D**: Defines a 2D scene. Takes a generator function `(view) => {}`; you add nodes to `view` and animate them with `yield*`.
- **Visual components**: [Circle, Rect, Polygon](components/shapes.md), [Txt](components/txt.md), [Code](components/code.md), [Layout](components/layout.md), [Img](components/img.md), [Line, Ray](components/line-ray.md), [LaTeX](components/latex.md), [Video](components/video.md), [SVG](components/svg.md).
- **Node**: Base class for all 2D nodes (position, scale, rotation, opacity, etc.). Use `createRef` from [core](core.md) to get a reference and animate.
